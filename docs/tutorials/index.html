Welcome to the SpaceRec tutorials

These tutorials explain how to use SpaceRec for dense spatial reconstruction from H&E histology. The current release focuses on a reproducible BRCA example with explicit execution stages and saved intermediate outputs.

Step-by-step guide for H&E-stained imaging data Run deconvolution, dense grid embedding, projection-head training, aggregation, and evaluation. Output reference Locate the generated CSV, H5, H5AD, checkpoint, and visualization files for each workflow stage. Environment notes Prepare the Python and R dependencies required by the notebook and RCTD deconvolution step. Model workflow overview Review how spot-level supervision is transferred to dense grid and polygon-level predictions.

Tested system

The full tutorial is GPU-oriented because the grid embedding and training stages use deep image features and projection-head fitting.

Python Python 3.10 with PyTorch, Lightning, NumPy, pandas, h5py, AnnData, Pillow, Matplotlib, Shapely, and timm.
R RCTD requires Matrix, Seurat, and spacexr for spot-level cell-type deconvolution.
GPU CUDA is recommended for Virchow2 grid embeddings and model training.

Quick start

Download the packaged artifacts, place data/ and results/ at the repository root, create the environment, and open the notebook. 

conda create -n spacerec python=3.10 -y
conda activate spacerec
pip install torch lightning numpy pandas scipy h5py anndata pillow matplotlib shapely timm

python -c "import spacerec.api as spacerec; print(spacerec.__file__)"
python -c "import torch; print(torch.cuda.is_available())"
Rscript -e "library(Matrix); library(Seurat); library(spacexr)"
Main notebook spacerec/notebooks/run_spacerec.ipynb
Dataset settings dataset = "brca", sample_id = "BREAST"
Input folders data/ and results/ at the repository root.
Output root results/<dataset>/<step>/
For smoke tests, lower max_patches, limit_spots, and max_epochs before running the expensive cells.

Workflow

The notebook is organized into five explicit execution stages. Each stage writes reusable files under results/brca/.

1 Deconvolution Estimate spot-level cell-type proportions.
2 Grid embedding Extract dense18 Virchow2 H&E features.
3 Training Fit expression and type projection heads.
4 Aggregation Map grids to polygon and cell predictions.
5 Evaluation Render type and expression checks.

Step 1: Deconvolution

RCTD estimates spot-level cell-type proportions from Visium counts and an annotated single-cell reference. These proportions supervise the model's type head during training.

x_s ~= l_s sum_k p_{s,k} r_k, p_{s,k} >= 0, sum_k p_{s,k} = 1
spacerec.deconv(
    dataset=dataset,
    visium_dir=visium_dir,
    sc_ref_h5ad=sc_ref_h5ad,
    output_dir=deconv_dir,
    output_csv=deconv_csv,
    sample_id=sample_id,
    annotation_column="Level1",
)

Step 2: Grid embedding

SpaceRec crops full-resolution H&E patches, extracts Virchow2 token features, maps tokens back to dense18 grids, and combines overlapping views with smooth spatial weights.

h_g = sum_v w_{g,v} h_{g,v} / sum_v w_{g,v}
spacerec.ge(
    dataset=dataset,
    he_image=he_image,
    positions_csv=positions_csv,
    scalefactors_json=scalefactors_json,
    thumbnail_png=thumbnail_png,
    output_dir=grid_dir,
    output_h5=grid_embedding_h5,
    model="virchow2",
    auto_select_gpu=True,
)

Step 3: Training

The training stage fits projection heads over the 6400-dimensional grid embeddings. One head predicts gene expression and the other predicts cell-type probabilities.

L = L_expr + lambda_type * [alpha L_conf + (1 - alpha) L_deconv]
spacerec.train(
    dataset=dataset,
    st_h5ad=st_h5ad,
    deconv_csv=deconv_csv,
    grid_embedding_h5=grid_embedding_h5,
    gene_list=gene_list,
    run_dir=train_dir,
    projection_dim=512,
    max_epochs=60,
    batch_size=4,
    auto_select_gpu=True,
)

Step 4: Aggregation

Grid-level predictions are transferred to polygon or cell regions by overlap area. This creates cell-level expression and cell-type probability tables.

xhat_c = sum_g a_{c,g} xhat_g / sum_g a_{c,g}
spacerec.agg(
    dataset=dataset,
    grid_expr_h5ad=grid_expr_h5ad,
    grid_type_csv=grid_type_csv,
    grid_predictions_h5=grid_predictions_h5,
    polygon_csv=polygon_csv,
    output_dir=aggregate_dir,
    valid_only=True,
    target_name="xen",
)

Step 5: Evaluation

The final cells render side-by-side checks for predicted and measured cell types and expression in selected spatial windows. 

spacerec.plottype(
    dataset=dataset,
    grid_type_csv=grid_type_csv,
    true_xen_type_csv=true_xen_type_csv,
    output_dir=evaluation_dir,
)

spacerec.plotexpr(
    dataset=dataset,
    grid_expr_h5ad=grid_expr_h5ad,
    true_xen_expr_h5=true_xen_expr_h5,
    output_dir=evaluation_dir,
    gene="EPCAM",
)
