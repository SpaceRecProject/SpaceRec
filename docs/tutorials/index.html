Overview

SpaceRec predicts dense grid-level gene expression and cell-type probabilities from H&E histology using Visium supervision.

1 Deconvolution Estimate spot-level cell-type proportions.
2 Grid Embedding Extract dense18 Virchow2 H&E features.
3 Train Fit gene and type projection heads.
4 Aggregate Map grid predictions to polygons/cells.
5 Evaluation Render type and expression comparisons.

Setup

Notebook spacerec/notebooks/run_spacerec.ipynb
Required artifacts data/ and results/ at the repository root.
Dataset setting dataset = "brca", sample_id = "BREAST"
Outputs results/<dataset>/<step>/
For smoke tests, set small values for max_patches, limit_spots, and max_epochs before running expensive cells. 
import spacerec.api as spacerec

dataset = "brca"
sample_id = "BREAST"

Step 1: Deconvolution

The deconvolution step uses Visium spot counts and an annotated single-cell reference to estimate spot-level cell-type proportions. The resulting deconv.csv supervises the model's type head.

x_s ~= l_s sum_k p_{s,k} r_k, p_{s,k} >= 0, sum_k p_{s,k} = 1
spacerec.deconv(
    dataset=dataset,
    visium_dir=visium_dir,
    sc_ref_h5ad=sc_ref_h5ad,
    output_dir=deconv_dir,
    output_csv=deconv_csv,
    sample_id=sample_id,
    annotation_column="Level1",
)

Step 2: Grid Embedding

The grid embedding step crops full-resolution H&E patches, resizes them for Virchow2, maps tokens back to dense18 full-resolution grids, and keeps grids inside the tissue boundary.

Feature layout
1280 token + 2560 same-patch tile + 2560 neighbor context = 6400 dimensions.
Patch settings
Default notebook flow uses 288px patches with stride 72.
h_g = sum_v w_{g,v} h_{g,v} / sum_v w_{g,v}
spacerec.ge(
    dataset=dataset,
    he_image=he_image,
    positions_csv=positions_csv,
    scalefactors_json=scalefactors_json,
    thumbnail_png=thumbnail_png,
    output_dir=grid_dir,
    output_h5=grid_embedding_h5,
    model="virchow2",
    auto_select_gpu=True,
)

Step 3: Train

Training fits projection heads over the 6400-dimensional grid embeddings. The gene head predicts grid expression and the type head predicts grid-level cell-type probabilities.

L = Huber(log(1 + xhat_s), log(1 + x_s)) + lambda [alpha L_conf + (1 - alpha) KL(p_s || phat_s)]
spacerec.train(
    dataset=dataset,
    st_h5ad=st_h5ad,
    deconv_csv=deconv_csv,
    grid_embedding_h5=grid_embedding_h5,
    gene_list=gene_list,
    run_dir=train_dir,
    projection_dim=512,
    max_epochs=60,
    batch_size=4,
    auto_select_gpu=True,
)

Step 4: Aggregate

Grid-level predictions are transferred to polygon or cell regions by area overlap. Cell-level expression and type probabilities are area-weighted averages over covered grids.

xhat_c = sum_g a_{c,g} xhat_g / sum_g a_{c,g}
spacerec.agg(
    dataset=dataset,
    grid_expr_h5ad=grid_expr_h5ad,
    grid_type_csv=grid_type_csv,
    grid_predictions_h5=grid_predictions_h5,
    polygon_csv=polygon_csv,
    output_dir=aggregate_dir,
    valid_only=True,
    target_name="xen",
)

Step 5: Evaluation

Evaluation renders side-by-side type and expression checks for selected windows. The notebook uses separate cells for type plotting and expression plotting. 

type_plot = spacerec.plottype(
