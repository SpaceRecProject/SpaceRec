Tutorials

Step-by-step notes for running SpaceRec on the packaged BRCA workflow, from H&E and Visium inputs to dense expression maps and cell-type predictions.

Overview

SpaceRec predicts dense grid-level gene expression and cell-type probabilities from histology using Visium supervision. The current tutorial follows the main notebook and keeps the workflow in five explicit stages.

  1. Step-by-step guide for H&E-stained imaging data
  2. Workflow overview
  3. Output reference

Tested system

  • Python 3.10 with PyTorch, Lightning, NumPy, pandas, h5py, AnnData, Pillow, Matplotlib, Shapely, and timm.
  • R packages: Matrix, Seurat, and spacexr for RCTD deconvolution.
  • CUDA is recommended for grid embedding and model training.

Quick start

Place data/ and results/ at the repository root, create the conda environment, and open the notebook. 

conda create -n spacerec python=3.10 -y
conda activate spacerec
pip install torch lightning numpy pandas scipy h5py anndata pillow matplotlib shapely timm

python -c "import spacerec.api as spacerec; print(spacerec.__file__)"
python -c "import torch; print(torch.cuda.is_available())"
Rscript -e "library(Matrix); library(Seurat); library(spacexr)"
Main notebook spacerec/notebooks/run_spacerec.ipynb
Dataset settings dataset = "brca", sample_id = "BREAST"
Input folders data/ and results/ at the repository root.
Output root results/<dataset>/<step>/

Workflow

  1. Deconvolution: estimate spot-level cell-type proportions.
  2. Grid embedding: extract dense18 Virchow2 H&E features.
  3. Training: fit expression and type projection heads.
  4. Aggregation: map grid predictions to polygons and cells.
  5. Evaluation: render type and expression checks.

Step 1: Deconvolution

RCTD estimates spot-level cell-type proportions from Visium counts and an annotated single-cell reference. These proportions supervise the model's type head during training.

x_s ~= l_s sum_k p_{s,k} r_k, p_{s,k} >= 0, sum_k p_{s,k} = 1
spacerec.deconv(...)

Step 2: Grid embedding

SpaceRec crops full-resolution H&E patches, extracts Virchow2 token features, maps tokens back to dense18 grids, and combines overlapping views with smooth spatial weights.

h_g = sum_v w_{g,v} h_{g,v} / sum_v w_{g,v}
spacerec.ge(...)

Step 3: Training

Training fits projection heads over the 6400-dimensional grid embeddings. One head predicts gene expression and the other predicts cell-type probabilities.

L = L_expr + lambda_type * [alpha L_conf + (1 - alpha) L_deconv]
spacerec.train(...)

Step 4: Aggregation

Grid-level predictions are transferred to polygon or cell regions by overlap area. This creates cell-level expression and cell-type probability tables.

xhat_c = sum_g a_{c,g} xhat_g / sum_g a_{c,g}
spacerec.agg(...)

Step 5: Evaluation

The final cells render side-by-side checks for predicted and measured cell types and expression in selected spatial windows. 

spacerec.plottype(...)
spacerec.plotexpr(...)
